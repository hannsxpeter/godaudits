# Dogfood audits

No external OSS dogfood audit currently meets the publication contract.

The required attribution, pinned source, focused scope, retained artifacts, and
honest misses and false positives are defined in
`docs/RELEASE-POLICY.md`. Add an entry to `index.json` only after every required
artifact is committed. Do not use a placeholder, an initialized unknown ledger,
or a self-audit to imply an external track record.
